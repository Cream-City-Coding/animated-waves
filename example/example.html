Animated Waves Component

A reusable web component for beautiful animated waves with customizable properties.

Basic Usage (Default Settings)

<animated-waves position="both" speed="0.5" height="10rem"></animated-waves>

Top & Bottom Waves with Content

<animated-waves 
    position="both" 
    wave-color="rgba(255,255,255,0.4)" 
    wave-height="80px"
    content-padding="40px"
    content-background-color="rgba(255,255,255,0.675)">
    <div class="card-content">
        <h3>Welcome to Our Website</h3>
        <p>This content is beautifully framed by animated waves above and below!</p>
        <p>The content area now has a matching background color.</p>
    </div>
</animated-waves>

📋 Available Properties

  • position: 'top' (default), 'bottom', or 'both' - Controls wave placement
  • wave-height: Height of each wave section (default: 100px) - Only for 'both' position
  • content-padding: Padding around content area (default: 20px) - Only for 'both' position
  • content-background-color: Background color of content area (default: matches wave-color) - Only for 'both' position
  • wave-color: Color of the waves (default: #1A237E)
  • background-color: Background color (default: transparent)
  • height: Height of the wave container (default: 15vh) - For 'top'/'bottom' only
  • speed: Animation speed multiplier (default: 1, higher = faster)
  • opacity-range: Min,max opacity for waves (default: 0.3,0.9)
  • wave-count: Number of wave layers (default: 4)
  • animation-style: smooth, linear, ease-in-out, bouncy, gentle (default: smooth)

💡 Usage Examples

Basic Usage

<!-- Basic top waves -->
<animated-waves></animated-waves>

<!-- Bottom waves only -->
<animated-waves position="bottom"></animated-waves>

Content with Waves

<!-- Top and bottom with content -->
<animated-waves 
    position="both" 
    wave-color="#1A237E" 
    wave-height="120px"
    content-padding="40px">
    <h2>Your Content Here</h2>
    <p>This content is framed by waves!</p>
</animated-waves>

Advanced Customization

<!-- Fully customized -->
<animated-waves 
    position="both"
    wave-color="rgba(255,255,255,0.4)" 
    wave-height="100px" 
    speed="2"

    opacity-range="0.2,0.8"
    animation-style="gentle"
    content-padding="60px"
